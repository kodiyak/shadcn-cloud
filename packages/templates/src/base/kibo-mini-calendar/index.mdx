# Kibo Mini Calendar

Customizable mini calendar component for React, built with Radix UI and Tailwind CSS.

## Features

- Customizable date selection
- Supports range selection
- Keyboard navigation
- Accessible with ARIA attributes
- Responsive design with Tailwind CSS
- Animations with Motion One

## Usage

```tsx
import { MiniCalendar } from "./index";

<MiniCalendar />
```
