# Shadcn Command

Command palette component for React, built with cmdk.

## Features

- Command palette with search
- Keyboard navigation
- Accessible with ARIA attributes
- Customizable design

## Usage

```tsx
import { Command } from "./index";

<Command>
  <div>Command content goes here</div>
</Command>
```
