# Shadcn Checkbox

Checkbox component for React, built with Radix UI.

## Features

- Customizable checkbox styles
- Keyboard navigation
- Accessible with ARIA attributes
- Responsive design

## Usage

```tsx
import { Checkbox } from "./index";

<Checkbox>Check me</Checkbox>
```
